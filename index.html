Web application

drinks

Drinks is a Single Page Application (SPA), that fetch data from a free API (TheCocktailDB.com). The web application is responsive and the user can implement search for a particular cocktail or drink. The response offers a list of drinks, each with instructions on how to prepare the drink. This project was implemented using the following technologies: Reactjs, Context, Axios, Materialze and JavaScript.
